MailSentry

Protecting businesses from
email fraud.

A Chrome extension that catches lookalike-domain scams in email — at the moment a payment decision is made. Fully client-side.

Try it yourself

~2 minutes. Same code as the GitHub repo.

Download mailsentry.zip
  1. Unzip mailsentry.zip.
  2. Open chrome://extensions in Chrome.
  3. Turn on Developer mode (top right).
  4. Click Load unpacked → pick the mailsentry/ folder.
  5. Accept the consent screen that opens in a new tab.
  6. Open Gmail — a banner appears above each email.
Optional: turn on live web checks
Without API keys, MailSentry still runs the sender, urgency, attachment, and QR checks — only link reputation and sender web-check are off. Both keys are per-user and free.
  1. Google Safe Browsing (checks links against known scam sites): go to console.cloud.google.com → enable Safe Browsing API → create an API key.
  2. Exa (checks unknown senders against the live web): sign up at dashboard.exa.ai → create an API key.
  3. Click the MailSentry icon in Chrome → paste each key into its field → Save. Keys are stored locally in Chrome storage and never sent anywhere except the respective API.
